# Data Sources

| Source | Tool | What it provides |
|--------|------|------------------|
| Calendar | `mcp_google_workspace_get_events` | Events for yesterday, today, tomorrow |
| Session history | `session_search` | Interaction patterns, mood signals |
| Memory | `chronicle_ask_about` | Known travel, projects, context |
| Location | Calendar events + chronicle | Current city, travel status |
